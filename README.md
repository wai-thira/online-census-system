# Online Census System

A web-based system developed for secure data collection and management. Built using PHP, MySQL, and front-end technologies.

## Features
- User authentication
- Input validation
- Basic security measures

## How to Run
- Clone the repository
- Set up XAMPP or a local server
- Import the database
- Run the project locally

## Database Design
The database schema is provided in `database_structure.sql`.

The schema was designed and implemented using MySQL (phpMyAdmin via XAMPP) and includes:
- User and admin authentication tables
- Household and family member relationships
- Audit logging for system actions
- Indexed fields for performance and integrity

Only the database structure is shared for security and privacy reasons.
